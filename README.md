# MessageBox

下载
npm i message-yang

import  element from'./element.js' 或 Common的var MessageBox = require('MessageBox')
的MessageBox（ '错误'， '消息'）
或者类似于vue
import MessageBox from 'MessageBox'
Vue.use(MessageBox)
